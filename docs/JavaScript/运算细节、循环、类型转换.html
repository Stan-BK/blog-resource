1、NaN、Infinity

Not a Number,除字符串相加外，NaN参与的运算结果都为NaN，且NaN不等于任何一个值（NaN也不等于NaN）。 除0外的任何数除以0都为±Infinity，正数为正，负数为负，0/0结果为NaN。

2、逻辑运算

与&& 或|| 非! 除undefined、null、NaN、""、0、false值以外，全部都是真。 与运算，遇真值便往后走，遇到假值或走到最后就返回当前值。 或运算，遇到假值往后走，遇真值或走到最后就返回当前值。

3、循环

for(var i = 0; i < 10; i++) { console.log(i); } 该for循环的步骤： 1、声明变量i= 0 2、if（i<10）{ console.log(i); } 3、i++ 4、重复2和3，如if（）中满足条件则继续，不满足停止循环 while循环即取for循环中的第二块（判断条件），上式改写为： var i = 0 while（i<10）{ console.log(i); i++; } 例题： for循环的（）中不能使用判断语句，{}中不能使用i++、i--，打印出0~100 答案： var i = 100; for（；i--；） { console.log(i); }

3、typeof（）

typeof函数可以获取数据的类型，比较特殊的一点是数组类型和null值（起初null值的存在就是为了表示空对象）获取到的结果是object。 typeof（array）=> object typeof（array）=> object typeof（function）=> function typeof（null）=> object typeof（undefined）=> undefined

4、类型转换

1、显式类型转换

Number（）、parseInt（）、parseFloat（）、String（）、toString（）、Boolean（）等 Number（）会尝试把值转化为数字类型。 parseInt（）会把数字转为为整形。 特殊点：parseInt('123ab12') => 123 parseInt('ab12') => NaN Number('123ab12') => NaN parseInt接受第二个参数作为进制转换。 String（）会把值转化为字符串类型，toString（）同理，toString（）也接受参数作为进制转换。

2、隐式类型转换

var a = '123'; a++;//先进行Number（a），再a++，结果为124 var b = '123'; ±b;//进行Number（b），结果为123和-123 总结： -*/都需要对数据进行Number（）后再运算，+视情况，如果是有字符串参与+运算则转化为字符串。

< >= <= ==会对数据进行Number（）转换，除非两者都是字符串，才进行ascii比较 特殊点：undefined == null => true

5、isNaN（）

isNaN内部会进行Number（）的隐式类型转换。
