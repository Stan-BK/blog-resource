1、CSS

CSS（Cascading Style Sheet）层叠样式表，作用是存储定义显示HTML元素样式的样式表。

2、CSS的引入方式

1、内联样式

直接写入到元素标签上

<p style=".."></p>

2、内部样式表

写入到<style>标签中

<style type="text/css">
	...
</style>

3、外部样式表

引入外部样式表资源，有两种方式。

  • link 属于 HTML，通过 <link> 标签中的 href 属性来引入外部文件，而 @import 属于 CSS语法，所以导入语句应写在 CSS 中，要注意的是导入语句应写在样式表的开头，否则无法正确导入外部文件；
  • 当 HTML 文件被加载时，link 引用的文件会同时被加载，而 @import 引用的文件则会等页面全部下载完毕再被加载；

1、通过<link>标签引入

<link rel="stylesheet" type="text/css" href="..."/>

2、导入方式

<style>     
	@import url(style.css); 
</style>

3、@import与<link>的区别

  • @import属于CSS语法，而<link>为HTML标签，可以进行标签属性的配置，使用<link>导入样式更加灵活。
  • 文档在解析到<link>标签时会对引入的CSS文件进行异步加载，@import则会在页面加载完毕后被加载。
  • <link>不存在兼容性问题，@import是于CSS2.1后推出的语法。
  • <link>可以通过JavaScript操作进行添加等操作，@import则没有相应操作方法。
  • @import导入的样式存在优先级问题，加载后它会处于样式表的最顶部（可以起到初始化样式的作用）。

4、优先级

内联样式 > 内部样式表 > 外部样式表

3、选择器

1、id选择器

id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式。id具有唯一性。 HTML元素以id属性来设置id选择器,CSS 中 id 选择器以 "#" 来定义。

<style type="text/css">
  #box1 {
    ...
  }
</style>
<div id="box1"></div>

2、class选择器

class 选择器用于描述一组元素的样式，class 选择器有别于id选择器，class可以在多个元素中使用，且一个元素可以拥有多个class。 class 选择器在HTML中以class属性表示, 在 CSS 中，类选择器以一个点"."号显示：

<style type="text/css">
  .box {
    ...
  }
</style>
<div class="box"></div>
<div class="box"></div>

3、标签选择器

标签选择器用于描述该标签类型的全部元素样式。

<style type="text/css">
  div {
    ...
  }
</style>
<div></div>
<div></div>

4、通配符选择器

通配符选择器会为所有标签类型的元素都设定样式，以一个"*"号表示：

<style type="text/css">
  * {
    ...
  }
</style>
<div></div>
<p></p>
<span></span>

5、属性选择器

属性选择器可以为所有具有该属性名或指定属性值的元素设定样式。多用于设置不同类型的input样式。

<style type="text/css">
  [id] {
    ...
  }
</style>
<div id="box1"></div>
<div id="box2"></div>

<style type="text/css">
  [id="box1"] {
    ...
  }
</style>
<div id="box1"></div>

6、派生选择器

通过依据元素在其位置的上下文关系来定义样式。

<style type="text/css">
  div .box{
    ...
  }
  .box.box1{
    ...
  }
</style>
<div>
  <div class="box"></div>
</div>

<div class="box box1"></div>

7、伪类选择器

依据元素当前状态进行选择，有:link、:active、:hover、:visited、:focus、:first-child、:nth-child(n)等

<style type="text/css">
  input:checked {
    ...
  }
</style>
<input type="checkbox" />当前选择框被选中时会有相应样式

8、CSS权重

CSS选择器有各自的权重，权重高的选择器展示样式的优先级高。CSS选择器的权重以256进制表示。

选择器 权重
* 0
标签，伪元素 1
class，属性，伪类 10
id 100
内联样式 1000
!important 正无穷

9、!important规则

在样式属性中加入!important可以增加样式的权重，即获得更高优先级。

<style type="text/css">
  #myid {
    background-color: blue;
  }
 
  .myclass {
    background-color: gray;
  }
 
  p {
    background-color: red /* !important */;
  }
</style>
//正常涂色
<p id="myid" class="myclass">1<p>	//蓝色
<p class="myclass">3<p>	//灰色
<p>3<p>	//红色
  
//如果为属性选择器p的background-color属性添加!important，所有p标签都会变为红色背景。

10、优先级

!important > id选择器 > class选择器 | 属性选择器 > 标签选择器 > 通配符选择器

11、匹配规则

浏览器在对CSS选择器进行匹配的时候，是以从右往左查找的顺序进行的。

<style type="text/css">
  <------------
  body div .box{
    ...
  }
</style>

4、动画

transform

transform属性允许旋转，缩放，倾斜或平移给定元素。这是通过修改CSS视觉格式化模型的坐标空间来实现的。 transform属性可取值详情参考：https://developer.mozilla.org/zh-CN/docs/Web/CSS/transform transform属性只对块级元素生效。通过定义一个或多个变换函数进行设置元素偏移，变换函数的执行顺序为从左到右。 常用变换函数有以下几个：

rotate()/rotate

rotate函数设置旋转（默认沿z轴旋转），角度单位有rad（弧度）、deg（度）、grad（百分比度）、turn（圈数）。该函数可以单独定义为属性。 rotateX函数沿X轴旋转，rotateY沿Y轴旋转， rotateZ沿Z轴旋转。 rotate3d 参数分别为（X轴偏移量、Y轴偏移量、Z轴偏移量、角度）。

scale()/scale

scale属性设置缩放比例（默认沿x轴和y轴缩放）。该函数可以单独定义为属性。 scaleX/scaleY/scaleZ分别沿X轴、Y轴、Z轴缩放。 scale3d 参数分别为（X轴缩放、Y轴缩放、Z轴缩放）。

skew()

skew属性定义一个元素在二维平面上的倾斜转换（默认沿X轴倾斜）。角度单位有rad（弧度）、deg（度）、grad（百分比度）、turn（圈数）。 skewX、skewY分别沿X轴、Y轴倾斜。

translate()/translate

translate属性定义元素平移（默认沿x轴平移），该函数可以单独定义为属性。 translateX/translateY/translateZ分别X轴、Y轴、Z轴平移。

transform-origin

该属性设置元素变化的原点。可以以百分比、绝对单位或关键字进行设置。该属性值默认为center。 设置一个值时，定义X轴方向； 设置两个值时，定义X轴和Y轴方向； 设置三个值时，定义X轴、Y轴和Z轴方向。

transition

该属性定义元素转变的方式，为以下四个属性的集合：

transition-property 设置元素变化方式
transition-duration 设置元素变化时间
transtion-timing-function 设置元素变化过程的表现
transition-delay 设置元素发生变化前后的延时

animation

该属性配置动画时间、时长以及其他动画细节，为一个配置集合，但该属性不能配置动画的实际表现，动画的实际表现是由 @keyframes规则实现。

animation-delay 设置延时，即从元素加载完成之后到动画序列开始执行的这段时间。
animation-direction 设置动画在每次运行完后是反向运行还是重新回到开始位置重复运行。
[

设置动画重复次数， 可以指定infinite无限次重复动画 | | animation-name | 指定由@keyframes描述的关键帧名称。 | | animation-play-state | 允许暂停和恢复动画。 | | animation-timing-function | 设置动画速度， 即通过建立加速度曲线，设置动画在关键帧之间是如何变化。 | | animation-fill-mode | 指定动画执行前后如何为目标元素应用样式。 |

细枝末节

1、浏览器默认的字体大小

font-size: 16px; 开发中常用的字体大小有12px、14px、16px。

2、设置字体倾斜

font-style: italic / oblique; 有些标签的字体没有italic属性，oblique可以强制让字体倾斜。

3、设置字体标识

font-family: "Times New Roman", Georgia, serif; 字体会从左往右进行选择，如果当前字体浏览器中不存在，则向后选择。带有空格的字体名，需用引号扩起。 兼容性好的字体蔟： font-family: "Hiragino Sans GB", "Microsoft Yahei", arial, \5b8b\4f53, "Helvetica Neue", Helvetica, STHeiTi, sans-serif;

4、行高

行高为一行占据的高度，组成为字体高度+上下边距。默认的行高为22px。 有四种设定值类型，分别为数字，长度，百分比和关键字（normal）。 如果以数字定义，则每个受行高影响的元素，其行高的求值根据元素自身的字体大小进行。 如果以长度定义（如em），则受行高影响的元素，其行高的求值根据其父元素的字体大小进行。 如果以百分比定义，则受行高影响的元素，其行高的求值根据系统默认字体大小进行。

5、绝对单位和相对单位

不会因为环境变化而改变自身度量的固定值单位就是绝对单位。会根据环境变化进行相应比例改变的单位就是相对单位。 CSS中的绝对单位为px，相对单位为em、rem。1em = 16px，1rem = 16px。 em相对于父元素计算，而rem相对于根元素计算（html）。em多用于设置字体。rem详解 设置字体大小为10px，即0.625em ：font-size：62.5%。

6、透明度

透明度属性opacity在IE中有兼容性问题，IE8及更低版本支持另一种写法： filter: Alpha(opacity=50);

7、设置容器内多行文本居中的方法

1、将容器diaplay设置为table 2、将容器内文本的display设置为table-cell 3、将容器内的文本的vertical-align设置成middle

8、box-sizing

设置盒子类型，常用值有border-box，会将边框和内边距都算入盒子宽高后绘制。需配置兼容性。

{
	box-sizing: border-box;
	-moz-box-sizing: border-box;
	-webkit-box-sizing: border-box;
	-ms-box-sizing: border-box;
	-o-box-sizing: border-box;
}

9、box-shadow

设置阴影，有六个配置项：水平偏移量、垂直偏移量、模糊距离、阴影尺寸、阴影颜色、阴影样式。需配置兼容性。

10、浮动

float属性可以让元素浮动，与周围元素形成环绕的形态。

  • 块级元素无法识别浮动元素的位置（即会占据其实际位置），所以会造成高度塌陷等问题。
  • 块级元素加了float属性后，会变成内联块级元素（inline-block）。
  • 清除浮动属性：clear: both;，需要在块级元素上才能生效。

11、伪元素

w3c规范的伪元素::before和::after，实际上也可以用:befor/:after伪类的写法去定义css。

  • ::before/::after添加样式必须有content属性。content属性也可以为url()，会引用相应资源。
  • 当两者在同一位置上时，::before在下层，::after在上层。
  • 伪元素的content属性可以设置值为对应元素中的属性，例：
<style type="text/css">
  p:before{
    content: attr(data-username);
  }
</style>
<p data-username="superatc">
  ,欢迎
</p>
//会显示文本：“superatc,欢迎”

12、边框圆角

border-radius属性可以设置边框的圆角，可以以百分比设置，也可以以像素为单位进行设置。需配置兼容性。

  • 设置圆：border-radius: 50%；
  • 设置半圆：border-radius：height/2 + px

13、背景

background-image: url() 设置背景图片 background-size: ...; 设置背景尺寸，可以为像素，也可以为百分比，也可以为相应属性值（如cover、contain等）。cover属性会按比例铺满整个盒子，当图片宽高比与盒子不符时会有部分被裁剪；contain属性会将整张图片按比例为放入盒子，当没有设置background-repeat: norepeat;时会重复。 background-repeat: ...; 设置背景重复。 background-position: ...; 设置背景位置，可以为像素，也可以为百分比。 background-attachment: ...; 默认值为scroll，背景会跟随盒子滚动；fixed值，背景固定，不跟随盒子滚动。 background: background-color background-image background-repeat background-attachment background-position/background-size;

14、设置logo

利用background属性设置logo而不是用img标签的原因：

  • img标签加载失败时的样式不雅观。
  • img标签资源加载时需要再次请求，而background属性的值具有缓存机制。
<style> 
  .wrap{
    width: 500px;
    height: 100px;
  }
  .logo{
    display: block;
    width: 500px;
    height: 0px;
    padding-top: 100px;
    background: url(./*.jpg);
    background-size: contain;
    background-attachment: scroll;
  }
</style>
<div class="wrap">
  <h1>	//h1标签有利于网页SEO
    <a href="" class="logo">Logo</a>
  </h1>
</div>

15、表格属性

caption-side: top/bottom; 设置表格标题在上方或下方。 border-collapse: collapse; 设置单元格边框合并，默认值为seperation。 table-layout: fixed; 设置单元格宽度一致，默认值为automatic。

16、BFC

Box Formatting Context BFC是一个独立的布局环境，其中的元素布局是不受外界的影响且不影响外界元素布局。 创建BFC的方法：

  • body元素
  • float: left/right
  • position: absolute/fixed
  • display: inline-block/table-cell
  • overflow: hidden/auto/scroll

BFC解决的问题：margin合并问题、margin塌陷、高度塌陷问题、浮动元素覆盖问题等。

17、层叠的特殊表现

  • 浮动元素的层叠被放置于非定位块元素和定位块元素之间。
  • 当一个非定位、浮动元素的背景设置了透明度（opacity），该元素也会隐式的创建一个层叠上下文，此时该元素会与文档中的定位块元素进行顺序层叠，且会置于其余非定位块元素以及浮动元素之上。
  • 每一个块级元素中的层叠子元素会基于其父元素的层叠上下文，与外界无关，即当A元素的层级大于B元素，A元素中的子元素无论层级高低，始终会置于B元素之上。
